This document describes the process of retrieving and analyzing the contents of a flash memory chip located in the PCB of a Sagemcom F@ST 3890v3 router. The memory chip in question is a Macronix MX30LF2G18AC as described in this datasheet: https://www.macronix.com/Lists/Datasheet/Attachments/8444/MX30LF2G18AC,%203V,%202Gb,%20v1.5.pdf
