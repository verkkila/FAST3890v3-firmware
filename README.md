# FAST3890v3-firmware
This repository contains documentation and tools related to the process of retrieving and analyzing the
contents of a flash memory chip in a Sagemcom F@ST 3890v3 home router.  
The documentation is written from the point of sharing information about the methods and techniques used, and any contents that a person doing similar projects could find useful.  

Other than that, the project has no explicit goal or purpose beyond curiosity and as a means of personal skill development.  
Documentation is located in the [doc](./doc/) folder of the repository, and tools are located in the [tools](./tools) folder.
